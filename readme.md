 1/What is the difference between var, let, and const?
 Answer:
 var=>Old way to declare variables (before ES6), It is Function scoped.
 let=>Introduced in ES6, It is Block scoped., it's value can be reassigned.
 const=>It is Used for constant values, It is also Block scoped and cannot be reassigned.

2/What is the spread operator (...)?
 Answer: Spread operator (...) expands elements of an array or properties of an object into individual elements.


3/What is the difference between map(), filter(), and forEach()?
 Ans:
 map()=>It is used to modify or transform elements and always returns a new array.
 filter()=> It is used to select elements that satisfy a condition and Returns a new filtered array
 forEach()=>It is used to run a function for each element and doesn't return a new array.
